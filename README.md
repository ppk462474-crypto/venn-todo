# 固定生活 80% 每日动作生成器

一个手机优先的个人生活管理小工具：通过「今日选择 / 今日 To-do / Happy Money」三个底部页面，生成温和、可执行的清单；还可以记录“快乐钱”。它是纯静态网页，不需要账号或后端。

## 使用

直接用浏览器打开 `index.html` 即可。建议在手机 Safari 中打开后，使用“添加到主屏幕”。

页面会自动识别日期与星期，工作日与周末会显示不同的模式标签；保存当天的模式、状态勾选和 To-do 完成状态。第二天会使用新的存储键，所以不会继承前一天的勾选。天气使用 Open-Meteo 的免费接口；若网络或城市查询失败，可在天气卡片中设置城市或尝试定位。

## 部署到 GitHub Pages

1. 新建一个 GitHub 仓库，并上传 `index.html` 和 `README.md`。
2. 在仓库 **Settings → Pages** 中，选择从分支部署。
3. 选择 `main` 分支与 `/ (root)` 文件夹，保存。
4. 等待 GitHub 提供页面链接后即可使用。

## 数据与修改

所有数据只保存在当前浏览器的 `localStorage`，不会上传到任何服务器。换设备、清浏览器数据或使用无痕模式都不会保留记录。

- 修改预算：展开页面底部的“调整预算与数据”，填写后保存。
- 修改固定星期规则：编辑 `index.html` 里的 `weekly()` 与 `defaultOptions()`。
- 增加模式：在 `modes` 对象中加入模式名称、原则 `p` 和任务数组 `t`。
- 清空数据：展开“调整预算与数据”，点击“清空所有本地数据”。

天气目前是无 API 的基础提醒版：出门前查看天气后，手动勾选是否需要带伞。
